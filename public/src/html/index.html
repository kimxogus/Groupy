Groupy

Groupy is Facebook open group post search helper.
It supports powerful post search and keyword alarm. Also it is cross-platform and easy-to-use!

Groupy can improve your convenience when use large Facebook open group. Here are some possible scenarios.
When you use large group, many new posts are written per day. Therefore, if you want to find specific topic of post then you have to do search. Facebook already has it's own search function within group, but it supports only simple search, so it is not enough for advanced search. (e.g. search from multiple groups, search from only title, etc.) Also there is no keyword alarm function in Facebook, so you have to monitoring the group every time if you want to read new interesting post immediately. This can be quite time consuming and annoying job. However, if you use Groupy you can register keyword and get alarm immediately when that keyword's new post is written.
